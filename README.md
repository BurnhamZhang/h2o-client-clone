
## 环境需求
* node `^6.4.0`
* npm `^3.0.0`


### 安装依赖包


```bash
$  npm install
```

### 打包

```bash
$ npm run build                   
```

### 开发
```bash
$ npm start                   
```

### 测试

**测试环境需要先打包**

```bash
$ npm run test                   
```

### 生产

**生产环境需要先打包**

```bash
$ npm run prod                   
```
